About Stanza

Stanza's mission is to make scheduling around the web/mobile seamless. Our smart "Add to Calendar" button is easy to embed in any business' website/mobile app and offers their audience a two-click process to sync future events with any calendar. Current customers include large sports franchises (NCAA, NBA, NHL, NFL) where we've seen 30M+ calendar event downloads since launch.

My Contributions

Built Stanza's analytics feature. Logic is written on the client along with data-visualization using HighCharts.js

Built Stanza's payment system.

Built internal tools for the bd team to help streamline the sales process.

Built the referral program at the time of Stanza's Button 2.0 launch.

Removed an outdated task runner and built grunt into an existing app to help speed up development time and to compress the build properly for deployment.

Worked with Stanza's lead UI/UX Designer, Jonathan Capecchi, and built Stanza's 'Timeline Campaign' feature which consists of a client-facing interface that allows the user to edit a responsive email-layout containing their upcoming events and details. The completed campaigin is then compiled on the server in Node.js and a responsive, transactional email is sent off to their subscribers with the embedded timeline of events.
